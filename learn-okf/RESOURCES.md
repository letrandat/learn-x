# Open Knowledge Format (OKF) Resources

## Knowledge

- [Blog Post: "How the Open Knowledge Format can improve data sharing" — Sam McVeety, Amir Hormati (Google Cloud Blog)](https://cloud.google.com/blog/products/data-analytics/how-the-open-knowledge-format-can-improve-data-sharing)
  The primary announcement introducing OKF, explaining the "fragmented context landscape" problem and the design principles.
- [GitHub Repository: GoogleCloudPlatform/knowledge-catalog/tree/main/okf](https://github.com/GoogleCloudPlatform/knowledge-catalog/tree/main/okf)
  The official open-source repository containing the OKF v0.1 specification, conformance criteria, and sample bundles (GA4, Stack Overflow, Bitcoin).
- [Gist: "LLM Wiki" — Andrej Karpathy](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f)
  The original source of inspiration explaining how LLMs are uniquely suited for maintaining and updating markdown-based wikis.
- [Blog Post: "Introducing the Google Cloud Knowledge Catalog" — Google Cloud Blog](https://cloud.google.com/blog/products/data-analytics/introducing-the-google-cloud-knowledge-catalog)
  Context on how Google Cloud integrates OKF into its broader Knowledge Catalog product.
- [GitHub Demo: OKF Tooling & MDCode Demo](https://github.com/GoogleCloudPlatform/knowledge-catalog/tree/main/toolbox/mdcode/demo)
  Reference code showing how to ingest OKF and feed it to agents.

## Wisdom (Communities)

- [GitHub Issues/Discussions: GCP Knowledge Catalog](https://github.com/GoogleCloudPlatform/knowledge-catalog/issues)
  The primary open-source channel for proposing extensions, filing issues, and engaging with other OKF adoptors.
