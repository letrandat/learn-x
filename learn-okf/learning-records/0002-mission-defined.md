# Mission Defined: Local OKF Wiki

Established a concrete mission based on user input: building a personal and professional local LLM wiki structured in the Open Knowledge Format (OKF). This grounds all future lessons in local, markdown-centric, and agent-queried wiki patterns rather than enterprise cloud catalogs.

- **Status**: active
- **Evidence**: User input expressing desire to build a local LLM wiki for home and work using OKF.
- **Implications**: Future lessons will focus on structural layout, hand-authoring files, writing a local parser or visualization script, and querying them using a local agent.
