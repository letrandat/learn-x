# Neogit & Git CLI Resources

## Knowledge

- [Repo: NeogitOrg/neogit](https://github.com/NeogitOrg/neogit)
  The plugin itself. README + issues for setup help.
- [Doc: neogit.txt (official)](https://github.com/NeogitOrg/neogit/blob/master/doc/neogit.txt)
  Complete reference for all popups, mappings, and configuration.
- [Book: Pro Git — Scott Chacon & Ben Straub](https://git-scm.com/book/en/v2)
  The canonical Git book, free online. Use for: understanding any git concept in depth. Start with Ch 2 (Git Basics) and Ch 3 (Branching).
- [Reference: git-scm.com/docs](https://git-scm.com/docs)
  Official git CLI docs. Use when you need the exact CLI flags.
- [Magit User Manual](https://magit.vc/manual/)
  Neogit's spiritual parent. The philosophy section explains why this UI style teaches git better than other GUIs.
- [Video: "Neovim Bankruptcy 02: Neogit" — Ioannis Canellos](https://www.youtube.com/watch?v=JwCKsEEHJBI)
  Walkthrough of installing and using neogit.
- [Wiki: Neogit Wiki](https://github.com/NeogitOrg/neogit/wiki)
  Community tips and workflows.

## Wisdom (Communities)

- [r/neovim](https://reddit.com/r/neovim)
  Active Neovim community. Good for neogit config questions and workflow discussions.
- [Neovim Discord](https://discord.gg/neovim)
  Real-time help. #plugin-discussion channel for neogit questions.
