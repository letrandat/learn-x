Diff commands to inspect state

CommandShows

Hunk-level staging

  1. In status buffer, press Tab on a file to expand
  2. Navigate to a hunk, press s to stage just that hunk
  3. CLI equivalent: git add -p (interactive patch mode)

Common patterns

Stage everything and commit

S          # stage all
c c        # commit

CLI: git add . && git commit

Stage only specific files

s          # stage file (repeat)
c c        # commit

CLI: git add file1 file2 && git commit

Unstage, fix, re-stage

u          # unstage
# edit file
s          # re-stage
c c        # commit

CLI: git restore --staged file && git add file && git commit
